# EU5 汎用攻略ガイド群 設計書

> 作成日: 2026-04-08
> 対象パッチ: 1.1（最新 1.1.10）

## 概要

EU5 の全国家に通用する基本攻略フレームワークを4ファイル構成で作成する。
既存の国家別ガイド（ブランデンブルク・オスマン・カスティーリャ・ハンガリー）から国家固有要素を除いた「共通骨格」を抽出し、政府・宗教・地域の分岐は専用ファイルに分離する。

## ファイル構成

```
eu5/
  eu5-universal-guide.md      ← 全国家共通の基本攻略（メインガイド）
  eu5-government-guide.md     ← 政府タイプ別の攻略差分
  eu5-religion-guide.md       ← 宗教グループ別の攻略差分
  eu5-regional-guide.md       ← 地域別Tips・おすすめ国家
```

HTML ビューア（`index.html`）にもカード4枚を追加する。

## テンプレート適用方針

- `eu5-universal-guide.md`: 既存13セクションテンプレートを「国家非依存版」にアレンジして適用
- 分岐3ファイル: テンプレート対象外（参照ガイド形式）。独自の見出し構成を使用
- 信頼性マーカー（`[src:]`・`（コミュニティ知見）`・`（未検証）`）は全ファイルで使用

## 出典方針

- スクリプト照合を行い、数値・条件には `[src:]` を付与する
- コミュニティ知見は Wiki・Reddit・Steam ガイドから収集し、マーカーで区別
- 国家別ガイドほどの網羅的スクリプト照合は不要だが、主要数値（エステート閾値、軍事比率、コスト等）は照合する

## Patch 1.2 更新候補マーカー

本文中に以下のマーカーを埋め込み、1.2 リリース時に grep で一括特定する:

```markdown
> ⚠ **1.2 更新候補**: {変更内容の概要}
```

### 1. eu5-universal-guide.md（全国家共通の基本攻略）

テンプレートのセクション構成を国家非依存にアレンジ:

```
# EU5 汎用攻略ガイド（Patch 1.1 時点）
> 目標: すべての国家に適用できる基本フレームワーク

## パッチ 1.1 の重要変更点（全国家影響）
  表形式: Complacency / 戦闘ダイス拡張 / 傭兵値下げ / 同化速度倍増 等
  + Patch 1.2 予告セクション（ベータ情報ベース）

## 最初の10分（Day 1 共通手順）
  番号付きリスト: ポーズ中にやること（内閣・エステート・外交・軍編成）
  自動化 ON/OFF の推奨初期設定
  ⚠ 1.2 更新候補: 官僚制スロットの初期設定が追加される可能性

## 序盤フレームワーク（開始〜1450 頃）
  Secure → Stabilize → Scale の3段階
  同盟選定の基本原則（脅威・近接性）
  最初の戦争タイミングの判断基準
  Complacency 回避のためのライバル設定（1.1 新規）

## 経済・内政の基本
  3本柱（課税・生産・貿易）の優先順位
  エステート忠誠度 50-70% ゾーンの管理
  食料備蓄と人口成長（1.1 仕様）
  建造物の優先順位（汎用推奨）
  ⚠ 1.2 更新候補: 官僚制システムで内政優先順位が変わる可能性

## 軍事の基本
  精鋭軍 vs 徴兵の使い分け
  推奨編成比率（歩兵6:騎兵2:砲兵2）
  兵站・補給線・戦争疲弊の管理
  傭兵の活用（1.1 で大幅値下げ）
  ⚠ 1.2 更新候補: 兵站見直しで運用が変わる可能性

## 外交の基本
  同盟・婚姻・属国の使い分け
  脅威システム（1.1）の活用
  → 政府タイプ別の外交差分は eu5-government-guide.md へリンク

## 中盤〜終盤への移行（1450〜1650〜）
  国家ランク昇格の条件と恩恵（County → Duchy → Kingdom → Empire）
  近接性管理（遠征 vs 隣接拡張）
  Advance（進歩）選択の一般指針（時代1〜6の汎用推奨）

## プレイ志向別ガイドライン
  表形式で4志向を比較:
  - 内政型（Tall）: 序盤優先=建造物・人口、拡張=最小限
  - 外交型: 序盤優先=婚姻・同盟網、拡張=同君連合・属国
  - 征服型（Wide）: 序盤優先=軍拡・CB確保、拡張=積極的
  - 植民地型: 序盤優先=海軍・探検、拡張=海外進出

## 分岐リンク集
  → eu5-government-guide.md（政府タイプ別）
  → eu5-religion-guide.md（宗教グループ別）
  → eu5-regional-guide.md（地域別Tips）
  → 既存国家別ガイドへのリンク一覧

## よくあるミス（全国家共通）
  10〜15 項目。エステート放置・Complacency 無視・兵站軽視 等

## 用語対照表
  汎用ガイドで使用した主要用語。完全版は localization-reference.md へリンク

## 出典
  一次情報（スクリプト・公式 Wiki）とコミュニティ情報を分離

### 2. eu5-government-guide.md（政府タイプ別）

```
# EU5 政府タイプ別攻略ガイド（Patch 1.1 時点）

## 概要
  5タイプの比較表（権力リソース・特徴・向いているプレイ志向）

## 君主制（Monarchy）
  権力リソース: Legitimacy
  強み: 王朝婚姻・同君連合
  弱み: 継承危機リスク
  序盤の優先事項 / エステート管理の注意点

## 共和制（Republic）
  権力リソース: Republican Tradition
  強み: 選挙で安定・優秀な統治者を選べる
  弱み: 王朝外交不可・Plutocracy 95+ が必要
  序盤の優先事項 / 選挙の活用法

## 神政国家（Theocracy）
  権力リソース: Devotion
  強み: 宗教改宗ボーナス・聖職者主導
  弱み: Spiritualist 95+ が必要
  序盤の優先事項 / 宗教戦略との連携

## ステップホード（Steppe Horde）
  権力リソース: Horde Unity
  強み: 略奪・国境急襲・戦争中の自動征服
  弱み: 統治者死亡で Unity -50・平時の維持が困難
  序盤の優先事項 / 定住化パスの判断

## 部族（Tribe）
  権力リソース: Tribal Cohesion
  強み: 農業コスト安い・初期の柔軟性
  弱み: 内閣長任命不可・中央集権化が遅い
  序盤の優先事項 / 改革パスの選択

## 政府変更の判断基準と手順
  変更コスト（安定度 -50 + 政府権力 -25）
  いつ変更すべきか / 変更しない方がいい場合

## ⚠ 1.2 更新候補セクション
  官僚制・新政府改革の影響予告

## 用語対照表
## 出典

## 実装方針

### 作業分割

- **バッチ1（本セッション）**: eu5-universal-guide.md の執筆 + index.html カード追加
- **バッチ2（本セッション）**: 分岐3ファイルを並列サブエージェント（sonnet x 3）で同時執筆
- CLAUDE.md のバッチ上限ルール（2バッチ/セッション）に収まる

### サブエージェント委任方針

分岐3ファイルは以下の指示で委任:
- テンプレート: この設計書の該当セクションに従う
- 出典: Wiki + コミュニティ知見。スクリプト照合は主要数値のみ
- 1.2 マーカー: 該当箇所に `⚠ **1.2 更新候補**` を埋め込む
- 変更の種類: 指定ファイルの新規作成のみ。既存ファイルは変更しない
- 共通 CSS/JS: 既存の index.html / viewer.html の仕組みをそのまま利用

### スクリプト照合対象

汎用ガイドで照合する主要数値:
- エステート忠誠度の閾値（反乱リスク・権力集中のライン）
- 軍事編成の基本コスト・戦闘修正値
- 国家ランク昇格の条件
- 政府変更コスト（安定度・権力）
- 宗教改宗の基本メカニクス
- Complacency の効果値
