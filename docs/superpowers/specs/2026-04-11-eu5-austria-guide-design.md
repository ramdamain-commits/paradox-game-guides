> ステータス: ドラフト

## 1. 概要

### 目的

EU5 のオーストリア（タグ: HAB）攻略ガイドを作成する。HRE 皇帝としての帝国運営とハプスブルク婚姻外交を両軸に据えた「外交大国プレイ」の攻略。

### ファイル

- **本体:** `eu5/eu5-austria-guide.md`
- **更新対象:** `index.html`（カード追加）、`eu5/localization-reference.md`（HRE 用語追加）

### パッチ基準

- Patch 1.1.10（2026-03-17）を本文の基準とする
- Patch 1.2「エキナデス」（ベータ）情報はパッチ差分セクションに分離

### 難易度想定

★★★（HRE メカニクスの理解が必要。eu5-regional-guide で「まず大国で覚える」推奨されている国）

### ターゲット読者

- **主対象:** EU5 の基本操作を覚えた中級者が HRE メカニクスを初めて学ぶ場面
- eu5-regional-guide が「HRE 小国はまずオーストリアで覚えてから」と誘導しているため、本ガイドは **HRE 入門としての役割も担う**
- HRE の汎用メカニクス（皇帝権限、帝国改革の仕組み）は本ガイド内で必要十分に説明し、eu5-government-guide の概要記述から詳細への誘導先とする
- 説明深度の基準: 「皇帝権限とは何か」レベルから書く。ただし EU5 の基本 UI 操作は既知と仮定

## 4. 既存ガイドとの関係

### 相互参照（双方向）

既存ガイドからオーストリアへの誘導パス、およびオーストリアから既存ガイドへの参照を双方向で設計する。

| 既存ガイド | 既存→オーストリア | オーストリア→既存 |
|-----------|-----------------|-----------------|
| ブランデンブルク | 皇帝との交渉セクションからリンク | HRE 運営セクションで「諸侯側の視点はブランデンブルクガイド参照」 |
| ハンガリー | 二重帝国ルート（`flavor_hun.550`）からリンク | 同イベントのオーストリア側視点で「ハンガリーガイド参照」 |
| オスマン | 反ハプスブルク連携の文脈でリンク | 対オスマン戦略で参照 |
| カスティーリャ | ローマ皇帝位イベント（`flavor_cas.84`）で言及 | 必要に応じて言及 |
| eu5-government-guide | HRE 概要から「詳細はオーストリアガイド」と誘導 | HRE 汎用メカニクスの概要説明後に government-guide へリンク |
| eu5-regional-guide | 「まずオーストリアで覚える」推奨からリンク | 整合性を確認 |

### 正本ルール（重複メカニクス）

同一メカニクスを複数ガイドで扱う場合の正本ルール:

| メカニクス | 正本 | 他ガイドでの扱い |
|-----------|------|----------------|
| HRE 皇帝権限・帝国改革 | **オーストリアガイド** | 概要+リンクのみ |
| 二重帝国ルート（HAB-HUN） | **各ガイドが自国視点で記述**。数値・条件はスクリプト出典を統一 | 矛盾発見時はスクリプトを再確認し両方を修正 |
| 婚姻・同君連合の汎用メカニクス | **オーストリアガイド 10.5b** が最詳。他は概要+リンク | — |
| 宗教改革の汎用フロー | **eu5-government-guide or 汎用ガイド**（調査後に確定） | オーストリア固有の対応のみ本ガイドに記載 |

### 差別化

- **ブランデンブルク:** HRE を「利用する側」→ オーストリアは「運営する側」で視点が逆
- **オスマン:** シチュエーション主導の統一戦略 → オーストリアは外交・帝国運営が主軸
- **ハンガリー:** 小〜中規模国の生存戦略 → オーストリアは大国としての帝国管理

## 6. 品質基準

- 既存4ガイド（特にオスマン・ハンガリー）と同等の品質・粒度
- 信頼性マーカー（`[src:]` / `（コミュニティ知見）`）を全数値に付与
- ローカライズ対照表で用語を確認してから記載
- HRE メカニクスは eu5-government-guide と重複しないよう、オーストリア固有の運用に焦点を当てる
