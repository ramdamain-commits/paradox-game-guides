**セクション2: パッチ変更点**

`## パッチ 1.1 / 1.1.10 でのオスマン関連変更点` テーブル形式。

以下を含める（ゲームスクリプト準拠）:
- AI 攻撃性のデフォルト低下 → オスマンのベイリク統一ペースに影響

- Patch 1.1.10: 建造物レベル0バグ修正、召集軍貼り付きバグ修正

**セクション3: 開始状況（1337年）**

`## 開始状況（1337年）` 以下3サブセクションを書く。

`### 「オスマンの台頭」シチュエーション`
- EU5ではオスマンは1444年に固定で存在しない。1337年に「オスマンの台頭」シチュエーションが自動発動
- アナトリアの `anatolian_beylik` 政府改革を持つ複数ベイリクが覇権を競う
- 最強ベイリクが `legacy_of_osman` → `legacy_of_the_ottomans` を取得しオスマン帝国を形成
- TUR以外が勝利した場合は RUM（ルーム・スルタン朝）への変形が可能
- コンスタンティノープル保有＋300都市以上で「オスマン帝国確立」宣言（`rise_of_the_ottomans.600`）

`### 周辺国との関係` テーブル形式。
- ビザンツ（BYZ）: シチュエーション開始時に安定度ペナルティ。外交要求（承認/金銭/廷臣引渡し）が可能
- カラマン（KAR）: 主要ライバル。`mehmed_beys_firman` イベント
- 他のベイリク: `anatolian_struggle_modifier` で相互に敵対意見
- ヴェネツィア（VEN）: コンスタンティノープル陥落後に衝突イベント発動（1450〜1551年）
- アルバニア: スカンデルベグ連鎖（人質要求の選択が展開を左右）

`### 初期の強み・弱み` テーブル形式。

強み:
- difficulty = 1（最も簡単なカテゴリ）
- 序盤から多方面に拡張可能（バルカン・アナトリア・中東）
- 固有イベント約207本で展開が豊富
- イェニチェリ（被ダメージ-10%）が中盤から利用可能

弱み:
- 1337年開始時はベイリクの1つに過ぎず、ゲーム開始直後は弱い
- ベイリク統一に失敗するとTURが形成されないリスク
- 後半の衰退メカニクス（イェニチェリの反乱、ティマール制崩壊）が厳しい
- 宗教的に欧州諸国と敵対しやすい（スンニ派 vs カトリック: `enemy` 評価）

**セクション4: Day 1（ポーズ解除直後）**

`## Day 1（ポーズ解除直後）`

`### 即座にやること` 番号付きリスト（5-8項目）。
1. 周辺ベイリクの勢力を確認、弱いベイリクを最初の征服対象に定める
2. ビザンツへの外交要求を検討（承認要求が安全、金銭要求はリスクあり）
3. 軍の集結位置を設定
4. 同盟候補の検討（他の大ベイリクとは同盟しない方がよい — 統一の障害になる）
5. ガーズィー進歩の取得を目指す（奴隷略奪効率+10%）
6. 内政: 階級特権の初期設定

`### 軍編成の基本方針` 2-3段落。
- 序盤は歩兵主力＋少数騎兵。統合軍備ボーナスを意識して多兵科混成
- イェニチェリ解禁後（Age 2進歩取得後）は主力をイェニチェリに切り替え
- アクンジュ（軽騎兵）は偵察・側面攻撃用に少数維持

**セクション8: 軍事ドクトリン**

`## 軍事ドクトリン`

`### イェニチェリ運用`
- 5段階進歩（Age 2〜6の各 janissaries_advance）で順次強化
- 被ダメージ-10%（強度・士気両方）が最大の強み
- 数量制限あり → イェニチェリ兵舎（janissary_barracks）を主要都市に建設
- 兵舎建設条件: 奴隷人口ベース、町・都市限定、地域人的資源+2.5%

`### アクンジュ運用`
- 発見時代（Age 3）解禁の固有軽騎兵
- 貴族人口連動の傭兵雇用可能
- 用途: 偵察、側面攻撃、占領加速

`### 召集軍（Levy）の扱い`
- Patch 1.1 で貴族召集軍の召集量が +20% → +50% に大幅上方修正
- ただし戦闘効率 +25% は削除 → 主力としてではなく数的優位の確保用
- 民兵法の戦闘効率も +25% → +10% に下方修正

`### 統合軍備（Combined Arms）ボーナス`
- Patch 1.1 で追加。多兵科混成で追加ボーナス
- イェニチェリ（歩兵）＋ アクンジュ（騎兵）＋ 砲兵の混成が理想

**セクション13: よくあるミス**

`## よくあるミス`

`### オスマン固有`
- ベイリク統一期に大ベイリクと同盟してしまう → 統一の障害に
- スカンデルベグ連鎖で人質を取る → 反乱リスクが高い（友好の方が安全）
- ミマール・スィナンの登用を見送る → 月収5倍は高いが建築ボーナスの価値が大きい
- イェニチェリの怒りイベントで改革を急ぎすぎる → 反乱を招く
- ティマール制の終焉を放置する → マーリカーネへの移行を早めに計画
- カリフ制主張の領土条件を見落とす → カイロ+アレクサンドリア+ダマスカス+エルサレム全て必要
- デヴシルメ廃止のタイミングを誤る → ニザーム・ジェディット移行と連動させる

`### EU5 全般`（ハンガリーガイドと同内容でよい — 以下を記載）
- 召集軍を常備軍として使ってしまう（維持費が高い、解散を忘れる）
- 統合軍備ボーナスを無視して単一兵科で編成
- 進歩の取得順を適当にしてしまう
- 建造物のレベル0バグ（1.1.10で修正済み）に気づかない
