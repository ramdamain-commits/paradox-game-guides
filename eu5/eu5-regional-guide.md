## 西欧（フランス・イベリア・ブリテン・イタリア）

### 地域特性

| 項目 | 内容 |
|------|------|
| 技術水準 | ゲーム内最高水準。制度（Institution）の普及が早い |
| 宗教 | カトリック主体。宗教改革後にプロテスタントが分出 |
| 政体 | 君主制・共和制が並立。イタリア都市国家は共和制 |
| 経済 | 交易・農業・工業がバランスよく発達。序盤から収入安定 |
| 外交 | 大国密集地帯。同盟・婚姻網の構築が生存の鍵 |
| 植民地 | ポルトガル・カスティーリャ・イングランドが植民地拡張の主役 |

### おすすめ国家

| 国家 | 難易度 | 学習目標 | 特記 |
|------|--------|--------|------|
| ナポリ（Naples） | ★☆☆ | 外交・内政の基礎 | 南伊の安定地帯。脅威が少なく初心者最適（コミュニティ知見） |
| ポルトガル（Portugal） | ★☆☆ | 植民地・海軍 | アフリカ沿岸探索から新大陸へ。植民地メカニクスの入門書 |
| フランス（France） | ★★☆ | 大国運営・内政拡充 | 初期から強大。外交的孤立に陥らないよう注意 |
| カスティーリャ（Castile） | ★★★ | イベリア統合・植民地 | アラゴン合同・レコンキスタ・新大陸の三方向を同時管理 |
| イングランド（England） | ★★☆ | 島国戦略・海軍 | 百年戦争後の再建から海軍覇権へ。ブリテン統一が中期目標 |
| ヴェネツィア（Venice） | ★★☆ | 交易共和国・海上帝国 | アドリア海貿易を軸に東地中海に影響力を持つ |

### 地域固有の注意点

- **制度の享受**（Institution Embracement）: 西欧は制度の発祥地になりやすい。他地域プレイ時にこの差が技術格差として表れるため、西欧との比較で自国の技術遅れを把握する習慣をつける
- **百年戦争の余韻**: 1337 年開始時点でフランスとイングランドは交戦状態に入りうる。序盤の介入タイミングを誤ると両国から敵視される
- **イベリア半島の分断**: カスティーリャ・アラゴン・ポルトガル・ナバラ・グラナダが並立。グラナダのレコンキスタ達成はカスティーリャの悲願であり、外部勢力がグラナダを支援すると外交摩擦の元になる（コミュニティ知見）
- **ブルゴーニュ問題**: ブルゴーニュ（Burgundy）はフランス・HRE の狭間で独立を保つ。どちらが吸収するかで中欧バランスが大きく変わる

> ⚠ **1.2 更新候補**: 官僚制（Bureaucracies）導入で、大国の行政スロット管理に新メカニクスが加わる。フランス・カスティーリャなど初期から広い領土を持つ国は特に影響を受ける可能性

## 中欧・神聖ローマ帝国（HRE）

### 地域特性

| 項目 | 内容 |
|------|------|
| 政治構造 | 皇帝（Emperor）と選帝侯（Elector）による連立体制 |
| 宗教 | カトリック主体。宗教改革でプロテスタント・改革派が台頭 |
| 小国密集度 | 世界最高水準。10 都市以下の小国が乱立 |
| 外交複雑度 | 帝国の権威（Imperial Authority）・選帝侯評価・加盟国との関係値が絡み合う |
| 技術 | 西欧に準じて高水準 |

### おすすめ国家

| 国家 | 難易度 | 学習目標 | 特記 |
|------|--------|--------|------|
| オーストリア（Austria） | ★★★ | 帝国政治・選帝侯管理 | 皇帝候補の最有力。HRE 全体を動かす醍醐味がある |
| ボヘミア（Bohemia） | ★★☆ | 帝国内政・中立外交 | 初期から選帝侯。宗教改革への対応が中期の核心 |
| ブランデンブルク（Brandenburg） | ★★☆ | 小国成長・外交生存 | HRE 小国から大国へ。のちのプロイセンの萌芽 |
| スイス（Switzerland） | ★★★ | 防衛特化・共和制 | 山岳防衛の要。傭兵供給拠点としての独自ポジション |

> HRE 小国は一般に初心者に非推奨。まず大国（オーストリア・ボヘミア）で帝国メカニクスを覚えてから小国に挑戦するのが推奨順序（コミュニティ知見）

### HRE メカニクスの要点

| メカニクス | 概要 |
|------------|------|
| 帝国の権威（Imperial Authority） | 皇帝の HRE 内での発言力。外国との戦争・小国保護など行動に影響 |
| 選帝侯（Elector） | 皇帝を選出する権利を持つ有力諸侯。評価を保つことが皇帝継続の鍵 |
| 帝国法（Imperial Laws） | 帝国内での戦争・吸収・宗教を規制する法体系。プレイヤーが皇帝なら変更可能 |
| 帝国議会（Imperial Diet） | 選帝侯・諸侯の集会。決議で帝国政策を変更 |
| 吸収（Integrate） | 従属国化した諸侯を直轄に編入。帝国権威・選帝侯評価への影響を考慮すること |

### 生存術（HRE 小国）

HRE 小国として生き残るには:
1. **選帝侯との婚姻・同盟**を最優先する
2. **帝国保護（Imperial Protection）**を盾に周辺大国の侵攻を抑制する
3. 序盤は拡張より**内政充実**（進歩・建造物）に集中し、人口・収入を底上げする
4. 宗教改革後は帝国内の宗派バランスに乗っかるか抵抗するかを早めに決断する（コミュニティ知見）

> ⚠ **1.2 更新候補**: HRE IO パネルのリデザインおよびメカニクス調整が予定されている。選帝侯管理・帝国権威の変動ルールが変わる可能性があり、このセクションの具体的な数値は正式リリース後に要確認

## 東欧（ポーランド・ハンガリー・ロシア・バルカン）

### 地域特性

| 項目 | 内容 |
|------|------|
| 政体 | 選挙制君主制（Elective Monarchy）が多い。ポーランド・ハンガリーが代表例 |
| 宗教 | カトリック（西部）・東方正教（東部・バルカン）・イスラム（オスマン支配圏）が接触 |
| 脅威 | 東からモンゴル・タタール系遊牧民。南からオスマン帝国 |
| 人口 | 西欧より低め。農業基盤が弱点 |
| 外交 | 大国間の緩衝地帯。同盟・婚姻で生存ラインを確保 |

### おすすめ国家

| 国家 | 難易度 | 学習目標 | 特記 |
|------|--------|--------|------|
| ポーランド（Poland） | ★★☆ | 外交・選挙制君主 | 北・東・南の大国に囲まれた外交の要。同君連合形成を狙える（コミュニティ知見） |
| ハンガリー（Hungary） | ★★☆ | エステート管理・軍事 | 詳細は eu5-hungary-guide.md を参照 |
| ロシア（Russia） | ★★★ | 遊牧民対策・大国化 | 初期は小国。ステップへの押し出し戦略が中期の核心 |
| セルビア（Serbia） | ★★★ | 正教圏・バルカン政治 | ⚠ 1.2 で正教メカニクス追加後に攻略内容が大幅変更見込み |

### 地域固有の注意点

- **選挙制君主の継承管理**: ポーランド・ハンガリーは君主の死後に次の王を選出する。候補者の能力・文化・宗教を事前に把握してエステートの支持を集めておくことが重要
- **遊牧民の騎兵速度**: 東方の遊牧民（タタール・ゴールデンホード系）は騎兵中心で移動速度が速い。城塞ラインを整備し、消耗戦で対処するのが基本（コミュニティ知見）
- **正教国との関係**: ロシア・セルビア・ワラキアは正教国。カトリック国からの宗教改宗外圧は評価を著しく下げる
- **バルカン争奪**: オスマンが南から、ハンガリーが北から圧力をかけるバルカン地帯。セルビア・ボスニア・ワラキアは強大な隣国に挟まれた状態でゲームが始まる

> ⚠ **1.2 更新候補**: 東方正教会の詳細メカニクスがパッチ 1.2 で追加予定。正教国（ロシア・セルビア・ワラキア等）の内政戦略・対外交渉ルールが大きく変わる。このセクションは 1.2 正式リリース後に全面見直し予定

## 中東・北アフリカ

### 地域特性

| 項目 | 内容 |
|------|------|
| 宗教 | スンニ派・シーア派イスラムが並立。宗派の違いが外交・戦争事由に影響 |
| 政体 | サルタン制・マムルーク制・部族連合など多様 |
| 貿易 | 紅海（Red Sea）・インド洋交易の要衝。交易路の支配が経済の核心 |
| 脅威 | 内部分裂（部族反乱）・十字軍・オスマンの拡張 |
| 資源 | 香辛料・絹・金の中継点。農業は砂漠地帯で低め |

### おすすめ国家

| 国家 | 難易度 | 学習目標 | 特記 |
|------|--------|--------|------|
| マムルーク朝（Mamluk Sultanate） | ★★☆ | イスラム政体・交易管理 | エジプト〜シリアを支配する中東最大の農業・商業大国。オスマンとの対決が中期の核心 |
| モロッコ（Morocco） | ★★☆ | イスラム拡張・北アフリカ統合 | イベリア半島への橋頭堡を持つ。カスティーリャとの緊張が特徴 |
| トレムセン（Tlemcen） | ★★★ | 北アフリカ小国生存術 | マグレブの中規模国。挟撃リスクを外交でいかに回避するかが腕の見せどころ |

### 地域固有の注意点

- **スンニ・シーア対立**: 宗派が異なる隣国との外交は「宗教的対立」モディファイアが加算され、関係構築が難しい。同盟形成前に宗派を確認すること
- **紅海交易の価値**: インド洋方向への交易路を抑えると莫大な収入を得られる。オスマン・ポルトガルが双方から交易路を狙うため、中東プレイでは海軍力の維持が不可欠（コミュニティ知見）
- **マムルーク朝の二正面**: エジプト方面の農業収入と、シリア〜アナトリア方向のオスマン圧力を同時に管理する必要がある。序盤に強力な同盟国を確保しておくと安定しやすい
- **砂漠補給**: 北アフリカ内陸部・アラビア半島は補給が厳しい地形が多い。大規模遠征前に兵站計画を確認すること

## 東アジア（中国・朝鮮・日本・東南アジア）

### 地域特性

| 項目 | 内容 |
|------|------|
| 人口・資源 | ゲーム内最大級。元の 91M 人口はダントツのトップ（コミュニティ知見） |
| 技術水準 | 中〜高。西欧との差は制度（Institution）普及速度で生まれる |
| 宗教 | 儒教・仏教・神道・イスラム（東南アジア）が並立。宗教改宗は評価への影響が大きい |
| 政体 | 帝国（中国系）・王国（朝鮮・ベトナム）・封建（日本）など多様 |
| 特殊 IO | 中華世界秩序（Chinese World Order / Suzerainty IO）。中国皇帝への朝貢が外交の軸 |

### コミュニティ強国ランキング（東アジア内）

| Tier | 国家 | 理由 |
|------|------|------|
| S | 元（Yuan / Great Mongol State） | 人口 91M、広大な領土、軍事力トップクラス（コミュニティ知見） |
| S | 朝鮮（Korea） | 高い技術基盤・安定した収入・比較的安全な立地（コミュニティ知見） |
| A | 明（Ming China） | 中国統一後の覇権国。ただし元の圧力で序盤が不安定になりやすい |
| B | 日本 | 統一前は群雄割拠。統一後は海外進出の道が開ける |

### おすすめ国家

| 国家 | 難易度 | 学習目標 | 特記 |
|------|--------|--------|------|
| 元（Yuan） | ★☆☆ | 大国運営・人口管理 | 初期から圧倒的な国力。大国プレイの感覚を掴むのに最適（コミュニティ知見） |
| 朝鮮（Korea） | ★★☆ | 技術・経済特化 | 隣接する元・日本との関係管理が求められる。技術研究の効率が高い（コミュニティ知見） |
| ベトナム（Vietnam系） | ★★★ | 中規模国の生存戦略 | 中国の属国状態から独立を維持する難しさが特徴 |

### 地域固有の注意点

- **中華 IO（朝貢システム）**: 中国皇帝への朝貢を怠ると「非朝貢」ペナルティが積み上がる。外交資源を割いて朝貢関係を維持するか、武力で打破するかの選択が東アジアプレイの核心（コミュニティ知見）
- **元の拡張抑制**: 元が序盤から急拡張すると周辺国が連合を組んで対抗してくる。元プレイでは慢心（Complacency）ペナルティを抑えつつ、外交的包囲を崩す順序が重要
- **日本の統一戦争**: 日本は 1337 年時点で多数の大名が割拠している。統一には内政安定と連続戦争を交互に組み合わせる粘り強いプレイが必要
- **東南アジアの分断**: マレー半島・インドネシア方面は多数の小国が乱立。ポルトガルやオランダ（将来的な勢力）の植民地化ターゲットになるため、早期の経済基盤強化が生存の鍵

> ⚠ **1.2 更新候補**: 東アジア方面の具体的な変更はオープンベータ情報には明示されていない（未検証）。パッチノート正式公開後に確認予定

## インド亜大陸

### 地域特性

| 項目 | 内容 |
|------|------|

| 人口 | 東アジアに次ぐ高密度。デリー・デカン高原方面が特に高い |
| 政体 | スルタン国・ラジプート王国・南インド帝国など多様 |
| 特殊 IO | ヒンドゥー世界秩序（Hindu IO）。ヴィジャヤナガルが中心的な役割 |
| 貿易 | インド洋交易の核心地帯。香辛料・綿布が主要輸出品 |

### コミュニティ強国ランキング（インド内）

| Tier | 国家 | 理由 |
|------|------|------|
| S | デリー（Delhi Sultanate） | 広大な北インド支配。大軍と高い農業収入（コミュニティ知見） |
| S | ヴィジャヤナガル（Vijayanagara） | 南インドの覇権国。ヒンドゥー IO の中心。経済力が高い（コミュニティ知見） |
| A | グジャラート（Gujarat） | 海上交易の要衝。港湾収入が安定 |
| A | ベンガル（Bengal） | 東インド・ベンガル湾の門番。農業と交易のバランスが取れている |

### おすすめ国家

| 国家 | 難易度 | 学習目標 | 特記 |
|------|--------|--------|------|
| デリー（Delhi） | ★★☆ | イスラム政体・大国拡張 | 北インドの覇者。周辺スルタン国の統合が序盤の目標（コミュニティ知見） |
| ヴィジャヤナガル（Vijayanagara） | ★★☆ | ヒンドゥー IO・南インド統合 | デカン高原南部を拠点に北進。豊かな農業収入が武器（コミュニティ知見） |
| グジャラート（Gujarat） | ★★★ | 海上交易・商業特化 | ポルトガルとの衝突が中期の難関 |

### 地域固有の注意点

- **多宗教管理**: ヒンドゥー・イスラム混在地域を支配すると宗教的不満（Religious Unrest）が発生しやすい。改宗政策か容認政策かを国家方針として早めに固めること
- **ポルトガルの侵入**: 中盤以降、ポルトガルがインド洋交易路に進出し、ゴア・コーチン方面に拠点を構えようとする。海軍力がなければ交易収入を奪われる（コミュニティ知見）
- **デカン・サルタン国群**: デリーとヴィジャヤナガルの間のデカン高原には複数のスルタン国が割拠。どちら側に引き込むかが中盤の外交の焦点
- **ヒンドゥー IO の活用**: ヴィジャヤナガルがヒンドゥー IO の中心であれば、ヒンドゥー諸国への文化的影響力ボーナスを活用して南インドを平和的に統合できる場合がある（未検証）

## 遊牧・ステップ（Nomadic / Steppe）

### 地域特性

| 項目 | 内容 |
|------|------|
| 政体 | ホード（Horde）型。略奪・騎兵突撃・移住が基本戦術 |
| 経済 | 農業ではなく略奪・牧畜が収入源。定住型の税収モデルとは根本的に異なる |
| 人口 | 低密度。広大な土地に少ない人口が分散 |
| 機動力 | 騎兵主体のため移動速度が極めて高い |
| 特殊 | 定住化（Settle Down）の判断がゲームを大きく変える分岐点 |

### おすすめ国家

| 国家 | 難易度 | 学習目標 | 特記 |
|------|--------|--------|------|
| ゴールデンホード（Golden Horde） | ★★☆ | ホード経済・略奪管理 | 東欧〜中央アジアをまたぐ大国。ロシア・ポーランドとの対決が序盤の焦点 |
| カザン・ハン国（Kazan Khanate） | ★★★ | 小ホード生存術 | ゴールデンホード分裂後の中規模勢力。拡張か定住かの選択が早期に迫られる |
| チムール朝（Timurids） | ★★★ | 中央アジア帝国再建 | チムール大帝の遺産を継ぐ。内部分裂への対処が最初の試練 |

### 定住型との根本的な違い

| 項目 | ホード | 定住型 |
|------|--------|--------|
| 主要収入 | 略奪・牧畜 | 税収・交易 |
| 軍事力 | 騎兵特化・高機動 | 歩兵・砲兵のバランス編成 |
| 内政 | 建造物建設が制限される場合あり | 制限なし |
| 進歩 | ホード専用ツリー | 汎用ツリー |
| 定住化後 | 定住型の内政モデルへ移行 | 変化なし |

### 定住化の判断基準

定住化（Settle Down）は不可逆的な判断であり、以下の観点を考慮する:
- 略奪対象が枯渇してきたタイミング（周辺に農業文明国が残り少ない）
- 人口増加・建造物建設が戦略上必要になった段階
- 定住型国家と長期的に競争するためのインフラ整備（コミュニティ知見）

> 定住化のタイミングは「遅すぎる失敗」より「早すぎる失敗」の方が回復困難。略奪収入が安定している間は定住化を焦らないことを推奨（コミュニティ知見）

## アフリカ・新大陸

### 地域特性

| 項目 | 内容 |
|------|------|
| 技術水準 | 初期は低い。ただし「技術後発」は即座の弱点ではなく、中期に拡大障壁になる |
| 宗教 | 独自宗教（アフリカ伝統宗教・マヤ・アステカ宗教など）。改宗圧力への対応が重要 |
| 人口 | アフリカは地域差大。中央アフリカは低い。マリ・ソンガイは相対的に高い |
| 特殊 | ヨーロッパ勢の植民地化と接触するまでは独立した世界観でプレイできる |
| 成長ポテンシャル | 開拓余地が広大。計画的な拡張で急成長が可能 |

### おすすめ国家

| 国家 | 難易度 | 学習目標 | 特記 |
|------|--------|--------|------|
| マリ帝国（Mali Empire） | ★★☆ | 西アフリカ統合・金交易 | サハラ以南最大の経済圏。金の産地を押さえると莫大な収入（コミュニティ知見） |
| エチオピア（Ethiopia） | ★★☆ | キリスト教アフリカ・東アフリカ統合 | 東方正教系キリスト教国。アフリカ内でのキリスト教連合形成が選択肢 |
| ソンガイ（Songhai） | ★★★ | 中規模アフリカ国の成長 | マリ後継。宗教多様性の管理と拡張の両立が課題 |
| アステカ（Aztec Empire） | ★★☆ | 新大陸完結型プレイ | ヨーロッパ勢が到達するまでの中米覇権。接触後の技術差が最大の難関 |
| インカ（Inca Empire） | ★★☆ | 南米覇権・山岳防衛 | アンデス山脈の地形を活かした防衛戦略。スペイン（カスティーリャ）との最終決戦が後半の山場 |

### ヨーロッパ勢との接触への備え

ポルトガルは 1415 年頃からアフリカ沿岸を南下し始め、1480 年代以降は新大陸への到達も視野に入る。この「接触」に備えるための戦略:

1. **技術投資の優先**: 接触前に軍事技術を可能な限り上げておく。技術差が縮まるほど初回接触時の被害が減る（コミュニティ知見）
2. **海軍整備**: アフリカ沿岸国はポルトガル海軍への対抗手段として、沿岸防衛網を早期に構築することを検討する
3. **内陸へ誘い込む**: 沿岸防衛が困難なら内陸への誘引と消耗戦を基本戦術にする（コミュニティ知見）
4. **同盟網**: 同宗教・同文化圏の隣国と事前に同盟を結び、ヨーロッパ勢の孤立化を図る

> ⚠ **1.2 更新候補**: アフリカ・新大陸固有のメカニクス変更はオープンベータ情報に明示なし（未検証）。バルカン新コンテンツのような地域追加が後続パッチでアフリカ・新大陸にも適用される可能性を注視
