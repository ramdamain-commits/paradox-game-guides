> オーストリア → ハプスブルク外交と神聖ローマ帝国運営を両立する序盤から帝国改革完遂までの整理。

## パッチ 1.1 / 1.1.10 でのオーストリア・HRE 関連変更点

ガイドの前提が古くならないよう、先にパッチ差分だけを抜き出しておく。

### Patch 1.1「ロスバッハ」での主要変更

| 変更 | 内容 |
|------|------|
| AI 攻撃性 | デフォルトで低下。HRE 内の諸侯間戦争が 1.0 系より少なくなる傾向。帝国権威の月次変動が安定しやすい |
| 神聖ローマ帝国 | 安定化調整。選帝侯 AI の投票行動が若干修正。ハプスブルク再選が通りやすくなった報告あり（コミュニティ知見） |
| 統合軍備（Combined Arms） | 多兵科混成で追加ボーナス。ランツクネヒト＋騎兵の混成編成の価値が向上 |
| 軍テンプレート | 編成テンプレートの保存・量産が可能に |
| 統治者を優遇（Favor the Ruler） | 最大税率 +5% を削除、**君主力**が 20% → **33%** に増加 |
| 民兵法（Common Militias） | 召集軍の戦闘効率が +25% → **+10%** に下方修正 |
| 貴族召集軍（Noble Levies） | 召集量 +20% → **+50%** に上方修正。戦闘効率 +25% は削除 |

### Patch 1.1.10（2026-03-17）

- 建造物がレベル 0 で残り、破産時に無限ループを起こす不具合を修正
- 召集軍が不正に部隊へ貼り付き続ける不具合を修正
- 正規部隊に意図せず追加されていた +100% 戦闘ボーナスを削除

> オーストリアは帝国防衛戦争で召集軍を多用するため、1.1.10 の召集軍修正は体感差が出やすい。

## 開始状況（1337年）

### 基本ステータス

| 項目 | 値 |
|------|-----|
| 国家ランク | 王国（Kingdom） |
| 政体 | 君主制（Monarchy） |
| 主要文化 | オーストリア（Austrian）、ドイツ語文化グループ |
| 宗教 | カトリック |
| 首都 | ウィーン（Wien） |
| 主要資源 | 農地・木材（ドナウ流域）。ハプスブルク領の鉱山収入あり |
| 属国 | シュタイアーマルク（STY）・チロル（TIR）等と同君連合の可能性（開始時の相続次第） |
| 難易度 | ★★★（HRE メカニクスの理解が必要） |

### 皇帝ステータス（最重要）

オーストリア（HAB）は 1337 年開始時点で**神聖ローマ帝国の皇帝**としてゲームを開始する。これが他の国との最大の差異であり、本ガイドの中心テーマとなる。

| 皇帝ボーナス（leader_modifier） | 値 | src |
|------|------|------|
| 外交容量 | +1 | [src: hre.txt, 139] |
| 最大外交官数 | +2 | [src: hre.txt, 140] |
| 外交射程 | +500 | [src: hre.txt, 141] |
| 要塞上限 | +1 | [src: hre.txt, 143] |
| 芸術家スロット | +1 | [src: hre.txt, 145] |
| 文化容量 | +1 | [src: hre.txt, 146] |
| 政府規模 | +1 | [src: hre.txt, 147] |
| 毎月威信 | +0.1 | [src: hre.txt, 148] |
| 毎月外交官（外交アクション頻度） | +0.5 | [src: hre.txt, 157] |
| 大国スコア免除閾値 | 50 | [src: hre.txt, 137] |

皇帝権限として「ラント平和令の強制」「宗教統一の強制」「不法領土の返還要求」「帝国軍の強化」「帝国の禁令 CB」などの専用アクションが使用可能になる [src: hre.txt, 外交アクション一覧]。

**帝国の権威（IA: Imperial Authority）** は 0〜100 の値で毎月変動し、HRE 運営の核心指標となる。IAが高いほど帝国法改革を推進しやすく、低いと選帝侯の反発や帝国解体のリスクが高まる。

## Day 1（ポーズ解除直後）

### 皇帝としての初動

HAB の Day 1 は通常の国家プレイと異なり、**HRE の現状把握**が最初の作業となる。帝国の権威（IA）・選帝侯の顔ぶれ・HRE 内の緊張状態を確認してから外交方針を決める。

1. **HRE パネルを開いて現状把握する**
   - 現在の帝国の権威（IA）を確認する。低い場合（50 未満）は序盤の拡張より安定化を優先する
   - 選帝侯の顔ぶれを確認する（公・選帝侯は最大 4 名、大司教・選帝侯は最大 3 名が上限 [src: hre.txt, 133-134]）
   - 選帝侯の対 HAB 好感度を確認し、低い選帝侯への外交を優先リストに入れる

2. **外交枠の配分計画を立てる**
   - 皇帝として外交官数が +2 されているため、通常より多く外交を展開できる [src: hre.txt, 140]
   - 優先順位: ①選帝侯への関係改善 ②ボヘミア王室婚姻の締結または確認 ③ハンガリーへの接触 ④オスマン方面への警戒外交

3. **ボヘミア（BOH）との王室婚姻を確認・強化する**
   - ボヘミアは HRE 内最有力の選帝侯候補かつ婚姻外交の最重要ターゲット
   - 王室婚姻（Royal Marriage）を締結すると marriage_union（結婚連合）が成立し、防衛戦争への共同参戦が確定する [src: hab-union-mechanics.md, marriage_union.txt:19-21]
   - 共通の後継者を生み出すことで将来的な Personal Union（同君連合）への布石となる

4. **ハンガリー（HUN）との関係構築を開始する**
   - ハプスブルク家の婚姻外交における最大のターゲット
   - 友好的な関係を維持しながら王室婚姻を狙う。同君連合成立には共通後継者が必要 [src: hab-union-mechanics.md, union.txt:270-295]
   - 開始直後に宣戦する必要はなく、まず関係を正の値に持ち込むことを優先する

5. **帝国の権威（IA）を上昇させる行動を確認する**
   - 帝国内が平和な状態が続くと +0.05/月の IA ボーナスが入る [src: hre.txt, 391-402]
   - 帝国自由都市が存在する場合は都市 1 つにつき +0.003/月 [src: hre.txt, 407-419]
   - 皇帝として HRE 防衛戦争に参戦して勝利すると大幅な IA 上昇が得られる [src: events/hre.txt, 706-721]
   - 序盤は HRE メンバーへの攻撃戦争を避ける（皇帝として HRE メンバーを攻撃すると −0.5/月のペナルティ [src: hre.txt, 369-387]）

6. **初期同盟候補を絞り込む**
   - 選帝侯との同盟は「帝位維持のための保険」として有効。ただし同盟選帝侯が属国化すると IA にペナルティが入るため従属させるつもりのない選帝侯と同盟する
   - バイエルン・ブランデンブルクなど、オーストリアの拡張先と被らない中規模諸侯が候補

7. **内政の初期設定を確認する**
   - 階級（Estate）の特権初期設定を確認し、貴族と聖職者のバランスを整える
   - 「栄光の一族」（外交官/月 +0.1、威信/月 +0.05）は age_1 の固有進歩。未取得なら早期取得を狙う [src: hab-advances.md, 1行]

8. **皇帝固有アクションの使用可否を確認する**
   - ラント平和令強制（`enforce_landfriede`）: HRE 内の諸侯間戦争を止めたい場面で使用
   - 不法領土の返還要求（`demand_unlawful_territory`）: 非メンバーが HRE 領を保有している場合に使用
   - 序盤は慎重に使用する。無用に使うと選帝侯との関係が悪化する場合がある

### HRE 入門：帝国権威と帝国改革の概要

オーストリアを語る上で HRE メカニクスの基礎を最初に整理しておく。

**帝国の権威（IA）の動き方**

帝国の権威は 0〜100 の値で毎月変動する。主な上昇・低下要因は次のとおり。

| カテゴリ | 主な要因 | 効果 |
|------|------|------|
| 上昇 | 帝国内平和 | +0.05/月 [src: hre.txt, 391-402] |
| 上昇 | 帝国自由都市（1都市ごと） | +0.003/月 [src: hre.txt, 407-419] |
| 上昇 | 防衛戦争勝利（hre.904） | 敵戦力/自戦力 × 5（上限 +10） [src: events/hre.txt, 706-721] |
| 上昇 | 諸侯の帝国宗教への改宗（hre.903） | 人口比 ×100（上限 +20） [src: events/hre.txt, 588-658] |
| 低下 | 皇帝が HRE メンバーを攻撃 | −0.5/月 [src: hre.txt, 369-387] |
| 低下 | 選帝侯不足（1名欠ごと） | −0.1/月 [src: hre.txt, 442-455] |
| 低下 | 属国選帝侯（1名ごと） | −0.05/月 [src: hre.txt, 456-471] |
| 低下 | 非 HRE 国によるメンバー併合（hre.905） | 最大 −20 [src: events/hre.txt, 744-756] |
| 低下 | 帝国防衛不参加（hre.909） | 強ペナルティ [src: events/hre.txt, 911] |

**帝国改革のロードマップ（概要）**

HRE の集権化は「皇帝快適性カウンター（`hre_emperor_comfort_policies_counter`）」を積み上げることで解放が進む [src: hab-hre-mechanics.md, Section 7]。

| ステップ | 制定する法律 | comfort カウンター | 解放内容 |
|------|------|------|------|
| 前提 | 金印勅書（golden_bull） | — | 帝国法体系の有効化 |
| 1 | 完全な帝国貢献（complete_imperial_contribution） | +1 | 累計 1 |
| 2 | 皇帝の後継者優先（emperor_successor_preference_policy）または帝位の世襲（erbkaisertum_policy） | +1 | 累計 2 → **帝国の権威の改善** が解放 |
| 3 | 選帝侯を統合（integrated_electors_policy） | +1 | 累計 3 |
| 4〜5 | さらに comfort 改革を追加 | +1〜2 | 累計 5 → **特権の剥奪・帝国の復活** が解放 |

最終段階の「帝国の復活（`renovatio_policy`）」に到達すると、帝国統一（全 HRE メンバーの外交的吸収）が可能になる [src: hab-hre-mechanics.md, 20_hre.txt:3068]。これがオーストリアプレイの長期目標の一つとなる。
