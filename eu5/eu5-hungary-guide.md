## パッチ 1.1 / 1.1.10 でのハンガリー関連変更点

ガイドの前提が古くならないよう、先にパッチ差分だけを抜き出しておく。

### Patch 1.1「ロスバッハ」での主要変更

| 変更 | 内容 |
|------|------|
| AI 攻撃性 | デフォルトで低下。オスマンの序盤拡張が 1.0 系より遅い傾向 |
| 神聖ローマ帝国 | 安定化。オーストリア・ボヘミアとの関係構築がしやすくなった |
| 統合軍備（Combined Arms） | 多兵科混成で追加ボーナス。騎兵主力でも歩兵・砲兵を混ぜるメリット増 |
| 軍テンプレート | 編成テンプレートの保存・量産が可能に |
| 統治者を優遇（Favor the Ruler） | 最大税率 +5% を削除、**君主力**が 20% → **33%** に増加 |
| 民兵法（Common Militias） | 召集軍の戦闘効率が +25% → **+10%** に下方修正 |
| 貴族召集軍（Noble Levies） | 召集量 +20% → **+50%** に上方修正。戦闘効率 +25% は削除 |
| イベント修正 | 「エルデーディ・タマーシュ」（`flavor_hun.270`）の選択肢とローカライズが修正 |
| イベント修正 | 「オーストリア系貴族の反乱」（`flavor_hun.480`）が 1.1 で発火可能に |

### Patch 1.1.10（2026-03-17）

- 建造物がレベル 0 で残り、破産時に無限ループを起こす不具合を修正
- 召集軍が不正に部隊へ貼り付き続ける不具合を修正
- 正規部隊に意図せず追加されていた +100% 戦闘ボーナスを削除

> ハンガリーは序盤の召集軍比重が高いため、1.1.10 の戦闘周り修正は体感差が大きい。

## Day 1（ポーズ解除直後）

### 即座にやること

1. **ブダに市場（Marketplace）を建設**
   - 交易収入の土台を最優先で確保

2. **内閣（Cabinet）を安定度（Stability）投資に割り当て**
   - 序盤の安定度維持が内政の土台

3. **不要な要塞（Fortification）を削除**
   - 維持費を浮かせて経済基盤を作る
   - 対オスマン防衛に重要なベオグラード方面の要塞は残す

4. **ライバル設定: セルビア、ボスニア、ワラキア**
   - 「州の請求（Claims on Province）」の開戦事由を確保しやすくする
   - ライバル国には領有権主張（Claim）を捏造しやすくなり、戦争の正当性を得やすい

5. **ポーランドとの関係改善（Improve Relations）を開始**
   - 王室間の婚姻（Royal Marriage）を結び、同君連合チャンスに備える

6. **金銀の輸出禁止を検討**
   - イベント「[国名]の金」（`flavor_hun.340`）が発火した際、「禁令を延長しろ」を選択
   - 国内で活用して制度（Institution）の伝播を加速

### 軍編成の基本方針（→ 詳細は[軍事ドクトリン](#軍事ドクトリン)を参照）

- ハンガリーの固有進歩「複合弓軽騎兵」で騎兵戦力 +20%。序盤から**騎兵比率を 40〜50%** に保つ
- 将軍は騎兵戦力に寄与する特性（Trait）を重視
- 1.1 の統合軍備（Combined Arms）ボーナスにより、歩兵・砲兵も混ぜるメリットが増加

## 序盤戦略（1337〜1400）

### フェーズ1: バルカン南下（1337〜1360）

**拡張優先順位:**

| 順位 | 対象 | 方法 | 備考 |
|------|------|------|------|
| 1 | セルビア・ボスニア | 州の請求（CB: 開戦事由）で征服 | 弱小国。最初の拡張先 |
| 2 | ワラキア・モルダヴィア | 属国化または辺境伯領（March）化 | オスマンとの緩衝地帯 |
| 3 | ダルマチア海岸 | 状況次第で征服 | ヴェネツィア領と接するため外交注意 |

**ポイント:**
- クロアチアは開始時点で同君連合下にあるため「征服」は不要。同君連合の統合（Integrate）を長期的に進める
- バルカンの山岳地帯は支配度維持が困難 → **属国を作る**のが効率的
- 属国は**辺境伯領（March）に変換**すると分権化ペナルティが軽減され、対オスマン防衛線としても機能する
- 属国は 2〜3 個まで。過剰な従属国（Subject）は**階級権力**のバランスを崩す
- セルビア征服後、バルカンを整理する前にオスマンには手を出さない

### フェーズ2: 内政強化と黒死病対策（1340〜1400）

> フェーズ1 と**並行して**進める内政タスク。

| 項目 | やること |
|------|---------|
| 病院建設 | 主要都市に**病院（Hospital）**を建設（1340年代までに） |
| 街道整備 | **街道（Road）**を全土に敷設。近接ペナルティを軽減 |
| 橋の建設 | **橋（Bridge）**を湖上に建設。ハンガリーの湖が近接障壁になっている |
| 鉱山開発 | 金銀鉱山（スロバキア・トランシルヴァニア）の開発を優先 |
| 製材所 | **製材所（Lumber Mill）**で建設資材を確保 |

## 中盤戦略（1400〜1500）

### ポーランド同君連合

- ポーランドとの**王室間の婚姻**と**高い評価（Opinion）**を維持し続ける
- **絶対にポーランドに宣戦しない**（同君連合の機会が消滅する）
- ポーランドの敵国と同盟しない（参戦要請でポーランドとの戦争に引きずり込まれるのを防ぐ）
- 同君連合成立後 → **統合（Integrate）**で外交コストを毎月消費し、約 50 年で完了

### ヴィシェグラード会議

イベント「[年]年のヴィシェグラード会議」（`flavor_hun.330`）で外交ボーナスが得られる。

- 選択肢「あの合意を成文化しよう」を選ぶとポーランドとの関係が強化される
- ポーランドが出席すれば第 2 回会議（`flavor_hun.331`〜`332`）につながる
- ポーランドが拒否した場合（`flavor_hun.18`）は関係に影を落とすが、致命的ではない

### オスマン対策

中盤の最大課題。正面衝突は避け、段階的に対抗する。

| 段階 | 行動 |
|------|------|
| 1 | バルカンの緩衝地帯を固める — ワラキア・モルダヴィアを辺境伯領化 |
| 2 | ベオグラード防衛を強化（イベント「ハンガリーの要」`flavor_hun.210` で「国費をさらに投入しよう」選択推奨） |
| 3 | 同盟網を構築 — オーストリア、ポーランド（同君連合済みなら不要）と共闘体制 |
| 4 | 騎兵主力の軍を整備 — 固有進歩のボーナスを最大活用 |
| 5 | **オスマンがアナトリアまたはマムルーク方面で戦争中**に、同盟軍と共に宣戦するのが最適タイミング |

### ドラゴン騎士団

イベント「ドラゴン騎士団」（`flavor_hun.410`）でオスマン対抗の騎士団を設立できる。

- 「いまこそドラゴン騎士団を結成するのだ」を選択すると他国にも参加要請が飛ぶ
- 対オスマン共闘体制の強化に有効

### 宗教改革への対応（1500 年前後〜）

イベント「[デブレツェン]の教会会議」（`flavor_hun.230`）でカルヴァン派拡大への対応を迫られる。

| 選択肢 | 効果 |
|--------|------|
| 「イエズス会を招致しよう」 | カトリック維持。安定路線 |
| 「改革の時だ！」 | カルヴァン派に転向。固有進歩「ハンガリー宗教改革」が解放される |

- カトリック維持なら固有進歩「カトリックの盾」（月間宗教的影響力 +0.1、自国宗教寛容度 +1）
- カルヴァン派転向なら「ハンガリー宗教改革」（国教会権力コスト -20%、自国宗教寛容度 +1）
- どちらでもない宗教なら「多宗教の地」（異端寛容度 +1、異教寛容度 +1）
- 宗教分岐は 3 つの固有進歩のうち 1 つだけ取得可能（排他）

## 終盤への移行（1500〜）

### 大国化の目標

- オスマンとの本格戦争（同盟軍と共に）。オスマンが他方面で消耗している隙を狙う
- ヨーロッパ列強としての地位確立
- オーストリア＝ハンガリー二重帝国の形成ルートも選択可能（ウィーン占領後のイベント `flavor_hun.550`）

> オーストリア側の視点は [オーストリアガイド](eu5-austria-guide.md) を参照。

### 注意: 「[国名]の進出」イベント（`flavor_hun.150`）

- オスマンに戦争で敗北するとハプスブルク家が介入し、ハンガリー領がオーストリアの中核州（Core）になる分裂イベント
- **このイベントが発生しないよう、オスマンに勝つか緩衝地帯を維持する**

### 首都避難計画（`flavor_hun.580`）

- オスマンの圧力が強い場合、首都をブラチスラヴァへ移す選択肢がイベントで提示される
- 防衛のために受け入れるか、ブダを死守するかはプレイスタイル次第

## 軍事ドクトリン

### 騎兵王国

ハンガリー最大の強みは固有進歩「複合弓軽騎兵」による**騎兵戦力 +20%**。これを活かす編成が鍵。

| 兵科 | 比率 | 備考 |
|------|------|------|
| 騎兵 | 40〜50% | ハンガリーの主力。上限ギリギリまで |
| 歩兵 | 40〜50% | 前線維持用 |
| 砲兵 | 10〜20% | 包囲（Siege）用。中盤以降に増やす |

### 召集軍（Levy）の扱い

| 項目 | 召集軍 | 正規（Regulars） |
|------|--------|------------------|
| コスト | 無料 | 高い |
| 補充 | 戦闘中は増援しない | 人的資源（Manpower）から補充 |
| 経験値 | 毎月減少、解散で喪失 | 維持される |
| 後半補正 | 絶対主義・革命時代に +10% | 特になし |

- 序盤は召集軍で数合わせ、中盤以降は正規軍中心へ移行
- 1.1 で貴族召集軍は量が +50% に増えた一方、戦闘効率補正は削除された

### 1.1 の統合軍備（Combined Arms）ボーナス

- 歩兵・騎兵・砲兵を混成すると追加ボーナス
- 騎兵偏重のハンガリーでも、少数の歩兵・砲兵を必ず混ぜることで恩恵を受けられる
- 軍テンプレート機能で騎兵主力編成を保存し、量産すると管理が楽

## 外交・同盟

### 必須同盟

| 国家 | 理由 |
|------|------|
| ポーランド | 同君連合候補。王室間の婚姻＋関係改善を最優先 |
| オーストリア | 対オスマン共闘。1.1 の神聖ローマ帝国安定化で関係構築しやすい |

### 推奨関係

| 国家 | 理由 |
|------|------|
| ボヘミア | 西方の安全確保。友好関係なら西を無視できる。イベント「カトリック信者からの支援要請」（`flavor_hun.180`）でボヘミア王位を得るルートもある |
| ヴェネツィア | 交易パートナー。序盤の物資輸入で制度伝播を加速 |

### 外交方針

- **君主力（Crown Power）**を高めて中央集権化を進める
- 1.1 で「統治者を優遇」の君主力補正が 20% → 33% に強化され、序盤の君主力確保がさらに重要に
- 王室間の婚姻は早期に結び後継者を確保する（王朝断絶を防ぐ）
- 摂政期間中は宣戦できないため、重要な戦争の直前に王が高齢なら開戦を急ぐ

## よくあるミス

### ハンガリー固有

| NG 行動 | 理由 |
|---------|------|
| ポーランドに宣戦する | 同君連合チャンスが永久に消滅。ポーランドの敵国との同盟にも注意 |
| 橋の建設を後回しにする | 湖による近接ペナルティで内政効率が大幅低下 |
| 歩兵だけの軍を編成する | 騎兵戦力 +20% ボーナスを無駄にしている |
| 従属国を作りすぎる | 辺境伯領化でペナルティ軽減できるが、3 個超は管理負担が重い |
| バルカン整理前にオスマンに挑む | 兵力・経済とも不足。オスマンが他方面で消耗しているタイミングを待つ |
| 病院建設を忘れる | 黒死病で人口壊滅 → 回復に数十年 |
| 君主力を軽視する | 中央集権化が遅れて中盤以降の選択肢が狭まる |
| 宗教改革を無視する | 固有進歩の宗教分岐を逃す。どの宗教にするか事前に方針を決めておく |

### EU5 全般

| NG 行動 | 理由 |
|---------|------|
| 複数戦線で同時に戦う | 兵力分散で各個撃破される |
| 人口の薄い土地に生産施設を先置きする | 労働力不足で稼働しない |
| 新領土の再建を急いで借金を膨らませる | 利息で経済が悪化 |
