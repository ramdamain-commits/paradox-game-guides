> 日本語は原則としてゲーム内ローカライズ準拠。未翻訳の箇所は英語を併記する。

## パッチ 1.1 / 1.1.10 でのカスティーリャ関連変更点

ガイドの前提が古くならないよう、先にパッチ差分だけを抜き出しておく。

### Patch 1.1「ロスバッハ」での主要変更

| 変更 | 内容 |
|------|------|
| AI 攻撃性 | デフォルトで低下。グラナダやモロッコの序盤拡張が 1.0 系より遅い傾向 |
| 統合軍備（Combined Arms） | 多兵科混成で追加ボーナス。テルシオ解禁前でも歩兵・騎兵混成のメリット増 |
| 民兵法（Common Militias） | 召集軍の戦闘効率が +25% → **+10%** に下方修正 |
| 貴族召集軍（Noble Levies） | 召集量 +20% → **+50%** に上方修正。戦闘効率 +25% は削除 |
| 軍テンプレート | 編成テンプレートの保存・量産が可能に |
| 統治者を優遇（Favor the Ruler） | 最大税率 +5% を削除、**君主力**が 20% → **33%** に増加 |

### Patch 1.1.10（2026-03-17）

- 建造物がレベル 0 で残り、破産時に無限ループを起こす不具合を修正
- 召集軍が不正に部隊へ貼り付き続ける不具合を修正
- 正規部隊に意図せず追加されていた +100% 戦闘ボーナスを削除

> カスティーリャは序盤のレコンキスタで召集軍を多用するため、1.1.10 の戦闘周り修正は体感差が大きい。

### Patch 1.2「エキナデス」（オープンベータ、正式リリース 2026年4〜6月予定）

> 以下はオープンベータおよび開発日記（Tinto Talks #100〜101）の情報に基づく。正式リリース時に変更される可能性がある。

## Day 1（ポーズ解除直後）

### 即座にやること

1. **ライバル設定: グラナダ、ナバラ、北アフリカの小国**
   - 州の請求（Claims on Province）の開戦事由を確保しやすくする

2. **アラゴンとの関係改善（Improve Relations）を開始**
   - イベリアの結婚（1450〜1500年発火）に備え、評価を高めておく
   - 王室間の婚姻（Royal Marriage）も早期に結ぶ

3. **ポルトガルとの関係維持**
   - リオ・サラドの戦い（`flavor_cas_rio_salado.1`）でポルトガルに同盟要請を送れる
   - 敵対しない限り友好を保つ

4. **ブルゴスに市場（Marketplace）を建設**
   - 交易収入の基盤を確保

5. **内閣（Cabinet）を安定度（Stability）投資に割り当て**
   - 序盤の安定度維持が内政の土台

6. **不要な要塞（Fortification）を削除**
   - 維持費を浮かせて経済基盤を作る
   - 対グラナダ前線のハエン方面は残す

### 軍編成の基本方針（→ 詳細は[軍事ドクトリン](#軍事ドクトリン)を参照）

- 固有進歩「レコンキスタ」（`cas_the_reconquista`）で陸軍士気 +15%。序盤から歩兵主力で積極攻勢が可能
- 1.1 の統合軍備（Combined Arms）ボーナスにより、騎兵・砲兵も少数混ぜるメリットが増加
- Age 3 でカスティーリャ騎槍兵（`cas_lanzas_de_castilla`）、Age 4 でテルシオ（`tercio`）が解禁される

| 4 | 勝利後、グアダルーペの聖母像（`flavor_cas_rio_salado.6`）と修道院建設が連鎖 `[src: flavor_cas_rio_salado.txt]` |

### フェーズ2: レコンキスタ — グラナダ征服

リオ・サラド勝利後、グラナダ方面に向かう。

**拡張優先順位:**

| 順位 | 対象 | 方法 | 備考 |
|------|------|------|------|
| 1 | グラナダ（GRA） | 聖戦 CB で征服 | レコンキスタの本筋。士気 +15% を活用 |
| 2 | アルヘシラス〜ジブラルタル | モロッコからの継続征服 | 海峡封鎖で北アフリカからの干渉を防ぐ |
| 3 | ナバラ方面 | 外交併合または州の請求 | 急がない。中盤でも間に合う |

**ポイント:**
- グラナダ征服後、モリスコ改宗イベント（`flavor_cas.1002`）が連鎖する。征服前に安定度を高めておく
- 「移民の誘致」（`cas_repoblaciones`）進歩で移住速度 +10%。征服地の再入植を加速
- グラナダの山岳地帯は支配度（Control）維持が困難 → 要塞を 1〜2 個残すか、駐留軍を配置

## 中盤戦略（1400〜1500）

### イベリアの結婚（最重要イベント連鎖）

1450〜1500年に後継者が女性で結婚可能、かつアラゴンまたはポルトガルに王朝候補がいる場合、「イベリアの結婚」（`flavor_cas.2200`）が発火する `[src: flavor_CAS.txt, flavor_cas.2200]`。これはカスティーリャ最大の分岐点。

| 選択肢 | 効果 |
|--------|------|
| 「アラゴンに婚姻の提案を送ろう」 | **推奨**。歴史的ルート。ARA との統合 → スペイン形成への最短路 |
| 「いや、ポルトガルにしよう」 | POR との統合ルート。大西洋植民に有利だがスペイン形成が遅れる |
| 「ほかの方法を模索する」 | 両方断る。非推奨（統合チャンスを逃す） |

**婚姻連鎖の流れ:**
1. `flavor_cas.2200`: 提案先を選択（ARA or POR）
2. `flavor_cas.2201`: 相手国が受諾/拒否を判断
3. `flavor_cas.2202`: 受諾 → 婚姻成立。**未選択の国（例: POR）が CAS を標的とした PU CB を取得**（`flavor_cas.2204`）
4. `flavor_cas.2203`: 拒否 → 別の解決策を検討

**注意:** 未選択の国（例: ARA を選んだ場合の POR）が CAS に対して同君連合 CB を得る（`flavor_cas.2204`〜`2205`）。防衛の準備をしておくこと。

### カトリック王の称号

1480〜1550年にカトリックで教皇庁と良好な関係にある場合、「カトリック王の称号」（`flavor_cas.22`）が発火する。

- 「称号を受け入れる」を選択 → 正統性大ボーナス + スペイン王の変数セット
- 拒否すると正統性大ペナルティ

### 異端審問所の分岐

1450〜1485年に「トルケマダとスペイン語の異端審問」（`flavor_cas.1000`）が発火。ゲーム最大の宗教分岐。

| ルート | 選択 | 連鎖イベント | 長期影響 |
|--------|------|-------------|---------|
| **審問所ルート** | 「トルケマダを支援しよう」 | 1001 アルハンブラ勅令 → 1002 モリスコ改宗 → 1003 アルプハラの反乱 → 1004〜1005 追放 | 安定度ペナルティが重いが、宗教的影響力大。聖職者満足度が高い |
| **寛容ルート** | 「この選択が招く結果は予期できない」 | 1006 イベリアの寛容（政府改革） | 安定度ペナルティなし。多文化・多宗教プレイに有利。聖職者満足度は低い |

**審問所ルートの連鎖詳細:**
- `flavor_cas.1001`「アルハンブラ勅令」: ユダヤ人追放。セファルディ文化が北アフリカへ大規模移民
- `flavor_cas.1002`「アンダルシア人の民の改宗」: スンニ人口75%をカトリック化。残りが北アフリカへ移民
- `flavor_cas.1003`「アルプハラの反乱」: 2〜10年後に自動発火。安定度極大ペナルティ
- `flavor_cas.1004`「アルプハラの戦い」: 1550〜1640年に追加の大規模反乱
- `flavor_cas.1005`「アンダルシア人の追放」: 最終段階。追放 or 審問所廃止の二択

（コミュニティ知見）審問所ルートは短期的な安定度リスクが高いが、宗教的影響力と聖職者満足度で長期的に有利。ただしアルプハラの反乱連鎖で安定度が壊滅するため、資金と軍を備えてから踏み込むこと。

## 終盤戦略（1500〜）

### コムネロスの乱

1500〜1600年に庶民（Commoners）の不満が高まると「コムネロスの組織」（`flavor_cas.1800`）が発火する。

| 段階 | ID | 内容 |
|------|-----|------|
| 前兆 | `flavor_cas.1800` | 弾圧 → 反乱軍増強 / 支持 → 貴族大ペナルティ |
| 勃発 | `flavor_cas.1801` | 反乱50%以上で発火。鎮圧か支持の二択 |
| 鎮圧 | `flavor_cas.1802` | カスティーリャ法廷強化ポリシー + 中央集権化 |
| 妥協 | `flavor_cas.1803` | コムネロス議会ポリシー + 地方分権化 |

（コミュニティ知見）鎮圧ルートが中央集権化の観点で有利。ただし反乱軍の鎮圧に軍を割かれるため、他国との戦争中に発火しないよう庶民階級の満足度管理に注意。

### スペイン形成

イベリア全域がキリスト教国または自国属国の支配下にあると「イベリア連合」（`flavor_cas.85`）が発火し、スペイン（SPA）を形成できる。

- 選択肢 a「伝統を尊重し、領土を統治しよう」→ 正統性大ボーナス + 中央集権化
- 選択肢 b「王権への中央集権化を図らなければ」→ 威信大ボーナス + 地方分権化

**前提条件:** アラゴンとの統合（イベリアの結婚経由）+ グラナダ征服 + ナバラ統合が必要。

## 軍事ドクトリン

### 士気の王国

カスティーリャ最大の序盤優位は固有進歩「レコンキスタ」（`cas_the_reconquista`）による**陸軍士気 +15%** `[src: country_CAS.txt, cas_the_reconquista]`。これを活かした歩兵主力編成が鍵。

| 兵科 | 比率 | 備考 |
|------|------|------|
| 歩兵 | 50〜60% | 前線維持の主力 |
| 騎兵 | 20〜30% | カスティーリャ騎槍兵（Age 3）解禁後に比率を上げる |
| 砲兵 | 10〜20% | 包囲（Siege）用。中盤以降に増やす |

### 海軍スタック

カスティーリャ海軍（`cas_castilian_navy`）で重船 +15%、海上プレゼンス +10% を得る `[src: country_CAS.txt, cas_castilian_navy]`。イベリア・ガレオン船と組み合わせると、EU5 屈指の海軍力になる。

### 召集軍（Levy）の扱い

| 項目 | 召集軍 | 正規（Regulars） |
|------|--------|------------------|
| コスト | 無料 | 高い |
| 補充 | 戦闘中は増援しない | 人的資源（Manpower）から補充 |
| 経験値 | 毎月減少、解散で喪失 | 維持される |

- 序盤のレコンキスタは召集軍で数合わせ。中盤以降は正規軍中心へ移行
- テルシオ解禁後は正規軍をテルシオに置き換えていく

## 外交・同盟

### 必須同盟

| 国家 | 理由 |
|------|------|
| アラゴン（ARA） | イベリアの結婚候補。王室間の婚姻 + 関係改善を最優先 |
| ポルトガル（POR） | リオ・サラド同盟。イベリアの結婚で ARA を選んだ場合、POR が敵対する可能性あり → 関係維持に注意 |

### 推奨関係

| 国家 | 理由 |
|------|------|
| フランス | 対アラゴンの牽制。ただしイタリア戦争（`flavor_cas.1400`）で敵対する可能性あり |
| 教皇庁（PAP） | カトリック王の称号（`flavor_cas.22`）に必要。関係を高く保つ |
| オーストリア | 中〜終盤の欧州政治で有用。カスティーリャとローマ皇帝位（`flavor_cas.84`）のイベントもある |

### 外交方針

- **君主力（Crown Power）**を高めて中央集権化を進める
- 1.1 で「統治者を優遇」の君主力補正が 20% → 33% に強化。序盤の君主力確保がさらに重要
- アラゴンとの婚姻は 1450 年前に確実に結ぶ（イベリアの結婚の前提条件）
- ポルトガルの空位時代（`flavor_cas.19`）で PU CB を得るチャンスもある

## よくあるミス

### カスティーリャ固有

| NG 行動 | 理由 |
|---------|------|
| アラゴンとの関係を放置する | イベリアの結婚（1450〜1500）に間に合わない。婚姻 + 評価を事前に整えておく |
| 異端審問所を無計画に受け入れる | アルプハラの反乱連鎖で安定度が壊滅。資金と軍の準備なしに踏み込まない |
| グラナダ征服後すぐに北アフリカへ拡張する | 征服地の統合（宗教・文化）が先。安定度が回復してから |
| 新大陸進歩の取得を遅らせる | ポルトガルに植民地を先行される。Age 3 で最速取得 |
| マゼラン遠征に金 250 を投じる余裕がないのに受ける | 国庫が枯渇する。経済的余裕を確認してから |
| コムネロスの乱を放置する | 議会管理を怠ると発火。他国との戦争中に重なると致命的 |
| ブルボン改革で植民地にインテンダンシーを導入しすぎる | 植民地の独立欲求が急上昇し、独立戦争に発展する |
| 海軍整備を後回しにする | 植民帝国は制海権が生命線。Age 2 でカスティーリャ海軍進歩を取得 |

### EU5 全般

| NG 行動 | 理由 |
|---------|------|
| 複数戦線で同時に戦う | 兵力分散で各個撃破される |
| 人口の薄い土地に生産施設を先置きする | 労働力不足で稼働しない |
| 新領土の再建を急いで借金を膨らませる | 利息で経済が悪化 |
