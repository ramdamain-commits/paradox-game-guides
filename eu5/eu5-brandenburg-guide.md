### Patch 1.1.10（2026-03-17、2026-04-05 確認時点の最新公開ホットフィックス）

- 建造物がレベル 0 で残り、破産時に無限ループを起こす不具合を修正
- 独立戦争と無関係な宣戦が独立戦争と誤表示される不具合を修正
- 召集軍が不正に部隊へ貼り付き続ける不具合を修正

> ブランデンブルク固有イベントの追加はないが、序盤は召集軍の比重が高いため 1.1.10 の戦闘周り修正は体感差が大きい。

## Day 1（ポーズ解除直後）

### 即座にやること

1. **「タンガーミュンデ城の拡張」イベント対応**
   - ポーズ解除から数ティックで発火しやすい
   - 威信（Prestige）を獲得できる
   - この威信を使って、皇帝に国家ランクの公国昇格を申請しやすくする

2. **東部の属国整理**
   - 支配度が低い州は直轄より属国委任が楽
   - 序盤の赤字と再建負担を抑えやすい

3. **庶民（Commoners）から「共同の土地（Communal Lands）」を剥奪**
   - 中央集権化の初手
   - **君主力**の回復を急ぎやすい

4. **政府改革で「専制政治（Autocracy）」を支持**
   - 君主力獲得に寄与する
   - 1.1 で「統治者を優遇」の君主力補正が強化され、序盤の君主力確保がさらに重要になった

5. **市場（Marketplace）を建設**
   - 交易収入の土台

6. **デンマークとの交易路を開設**
   - 開始時同盟をそのまま経済にもつなげる

### Day 1 のディザスター発火

- **「ブランデンブルクのヴィッテルスバッハ家」**イベントが発生
- **「ヴィッテルスバッハ家の怠慢」**デバフを受ける
  - 支配度獲得速度の低下
  - 正統性獲得の低下
  - ディザスター解決まで響く重いペナルティ

### 軍編成の基本方針

- 序盤は拡張より内政安定を優先
- **召集軍（Levy）**は無料だが、戦場で増援せず、解散で経験値も失う
- 同盟国に好感度を使って参戦してもらうのが基本
- 余裕資金があれば傭兵で穴を埋め、本軍は決戦に集中
- 1.1 で貴族召集軍は量が増えた一方、質の補正は落ちた

## 序盤戦略（1337〜1500）

### フェーズ1: 「ブランデンブルクの混乱」克服

Patch 1.1 で追加された固有ディザスター。旧イベント群を一体化したもので、王朝問題と地方支配の両方を処理させられる。

#### 構成要素

| チェーン | 内容 | 時期 |
|----------|------|------|
| ヴィッテルスバッハ家の怠慢 | 支配度獲得速度と正統性に重い悪影響 | 開始直後 |
| 偽ヴァルデマール | 正統性に対する詐称者イベント | 序盤 |
| 泥棒男爵 | 地方貴族の実力支配と反乱 | 1365 年以降 |

#### ディザスター終了条件

スクリプト上は次のいずれかで終了する。

- `bra_wittelsbacher_investment_var >= 100`
- `bra_wittelsbacher_neglect_var <= -100`
- 王朝決着により上記変数自体が無効化される

つまり、**-20 では終わらない**。放置ルートを取るなら、放置値を **-100** まで積む必要がある。

#### 戦略A: 投資ルート

1. 支配強化イベントでは投資寄りの選択肢を優先
2. `bra_wittelsbacher_investment_var` を 100 まで積む
3. 終了後に支配度回復と直轄化を進める

#### 戦略B: 放置・王朝転換ルート（ホーエンツォレルン獲得）

ホーエンツォレルンへの王朝転換にはスクリプト上**少なくとも二系統のルート**がある。

**ルート B-1: neglect -100 → 皇帝介入チェーン**

1. ヴィッテルスバッハ家関連イベントで意図的にマイナス側を選ぶ
2. `bra_wittelsbacher_neglect_var` を **-100** まで下げる
3. 皇帝介入イベントチェーンが発火し、王朝交代の分岐に入る
4. 出典: `turmoil_in_brandenburg.txt:2040-2177`

**ルート B-2: 皇帝従属下の月次イベント**

1. 皇帝がブランデンブルクを従属国または同君連合として支配下に置いている場合に発動
2. 月次チェックで王朝交代イベントが発火する
3. neglect -100 を積む必要がない別ルート
4. 出典: `country_monthly.txt:57`, `turmoil_in_brandenburg.txt:2419-2515`

> **注意**: どちらのルートでも「必ず何年で到達」とは言い切れない。コミュニティ報告（Steam Discussions ベース）では、オーストリアが皇帝のときに B-1 が噛み合いにくい例があるが、これはスクリプト断定ではなく報告ベースの知見。
>
> **1.1 での変更点**: ホーエンツォレルンのキャラクターは皇帝の主文化ではなく**東フランケン文化**を持つように変更された。また、皇帝の親族選出が常に**成人の後継者を優先**するように修正されている。

#### ディザスター中の注意

- 投資ルートと放置ルートを途中で混ぜると終着点が遅れやすい
- 泥棒男爵対処前に外征を広げると内政が崩れやすい
- 支配度が戻る前に属国を無理に直轄化しない

### フェーズ2: 序盤の拡張

#### 拡張の原則

| ルール | 理由 |
|--------|------|
| **属国化を優先** | 支配度不足の直轄領は負担が大きい |
| **北方から外側へ拡大** | 弱い相手から刻みやすい |
| **領土紛争（Border Dispute）を活用** | 併合コストの軽い開戦事由を確保しやすい |
| **帝国内の過度な拡張を避ける** | **敵対心（Antagonism）**急増から包囲網につながる |

#### 征服時の注意

- 新領土の負債や食糧不足まで一緒に抱える
- 重要施設の再建を優先し、借金拡大を避ける
- 直轄化より属国統合のほうが楽な局面が多い
- 1.1 以降、帝国内の CB なし宣戦はさらに重い

#### やってはいけないこと

- 神聖ローマ帝国内での急速な拡張
- 支配度不足のままの直接併合
- 皇帝との関係悪化

## 中盤戦略（1500〜1650）

### プロイセン建国

#### 建国条件

| 条件 | 詳細 |
|------|------|
| 文化 | ドイツ系文化グループ |
| 領土 | **プロイセン地域（Prussia Area）の 75% 以上**を直接所有（`required_locations_fraction = 0.75`）。全州は不要 |
| 政体 | **軍事修道会（Military Order）でないこと**（君主制・共和制いずれも可。建国時に君主制でも共和制でもない政体の場合のみ君主制へ変更される。共和制はそのまま維持） |
| 状態 | 平和状態 |
| 階級特権 | 「土地権利（Land Rights）」保有 / 「強力な司教区（Powerful Dioceses）」未保有 |
| 聖職者権力 | **20% 未満**であること |
| 貴族権力 | **20% 超過**であること |

> **注意**: 聖職者権力 < 20% と貴族権力 > 20% の条件は見落としやすい（コミュニティ報告ベースでは建国できない原因として頻出）。身分管理セクションで意識的にこの数値を目標に調整すること。
>
> **出典**: 建国条件は `00_formable_countries.txt:1005-1049`（PRU_f 定義）に基づく。

#### 建国までのステップ

1. ディザスター「ブランデンブルクの混乱」を解決する
2. 属国管理で領土と支配度を整える
3. ドイツ騎士団からダンツィヒを確保し、北東進出の足場を作る
4. プロイセン地域（Prussia Area）の **75% 以上**を直接所有する
5. 皇帝との関係を整え、王国化交渉を有利にする
6. プロイセン建国後に軍制と進歩を本格化させる

#### 王国化は二択ではなく三分岐

「皇帝との関係が良ければ王国、悪ければ公国のまま」と単純化しないほうがよい。イベント上は以下の分岐がある。

- **妥協案**: 「King in Prussia」を名乗ることだけ認められ、後で帝国内昇格議題が開く
- **正式承認**: 帝国内でそのまま王国へ昇格
- **拒否**: 皇帝決定を受け入れて公国のまま残るか、帝国を離脱して王国化するかを選ぶ

> 皇帝との関係改善は承認率を上げる材料にはなるが、結果は関係値だけで固定されない。

### 宗教改革と改宗

#### プロイセン建国と宗教

- 主要宗教の変更コストは**安定度 50**（出典: `00_hardcoded.txt:281-283`）
- プロテスタント改宗は歴史ルート寄りだが必須ではない
- **白鳥騎士団**はカトリック限定（1420-1460）なので、改宗タイミングは白鳥騎士団の取得後にすること
- 宗教改革は通常 1500 年前後に発生するため、白鳥騎士団の期間とは重なりにくい

#### 帝国宗教法への影響

- 初期の選帝侯宗教法はカトリック前提
- 改宗すると選帝侯運用に影響する
- 1.1 で革命国家は皇帝選挙資格を失うようになった

### 神聖ローマ帝国の活用

#### ブランデンブルクに関係する主要メカニクス

| 項目 | 内容 |
|------|------|
| 選帝侯 | ブランデンブルクは 7 選帝侯の 1 つ |
| ランク昇格 | 帝国内では皇帝の許可が重い |
| 敵対心 | 帝国内での併合は**敵対心**が急増しやすい（※具体的な倍率はパッチノート原文で明示されていないため、ゲーム内ツールチップで確認推奨） |
| 開戦事由制限 | CB なし宣戦は重いペナルティ（※同上、具体倍率は要ゲーム内確認） |
| 帝国防衛 | 外部勢力の侵入時は皇帝が介入しやすい |
| ドイツ宮廷語 | 1.1 以降、選挙影響が +20 |

#### 皇帝を目指すべきか

基本的には不要。まずはプロイセン建国と北東の基盤確保を優先したほうが安定する。

- 皇帝業務は兵力と外交枠を食う
- 友好的な他国を皇帝にしたほうが交渉しやすい
- 拒否分岐まで考えるなら、帝国に残るか離れるかを自分で選びやすい

## 終盤への移行（1650〜）

### 大国化の目標

プロイセン建国と王国化を達成した後は、固有進歩の軍事ボーナスを最大限に活かした拡張フェーズに入る。段階 V の**軍隊の専門化**（陸軍士気 +10%）と段階 VI の**ガチョウ足行進**（歩兵戦力 +20%）がプロイセンの強みを決定づける。

北ドイツの支配を固めつつ、帝国内外の情勢に応じて影響圏を広げる。帝国に残る場合は選帝侯としての影響力を活用し、離脱した場合は自由な拡張路線を取れる。いずれのルートでも、正規軍中心の軍制への移行と経済基盤の強化が後半の鍵となる。

## 軍事ドクトリン

### プロイセン擲弾兵

Patch 1.1 で追加された固有ユニット。絶対主義時代にはさらに強化版の**絶対主義時代の擲弾兵**が解禁される。固有進歩の**ガチョウ足行進**（歩兵戦力 +20%）と組み合わせることで、後半のプロイセン歩兵は他国を圧倒する戦闘力を発揮する。

### 召集軍（Levy）の扱い

| 項目 | 召集軍 | 正規（Regulars） |
|------|--------|------------------|
| コスト | 無料 | 高い |
| 補充 | 戦闘中は増援しない | 人的資源から補充 |
| 規律 | -10% ペナルティ | なし |
| 経験値 | 毎月減少、解散で喪失 | 維持される |
| 対正規戦 | 序盤はかなり脆い | 安定 |
| 経済影響 | 出身州の食料と原材料を圧迫 | 小さい |
| 後半補正 | 絶対主義時代・革命時代に +10% | 特になし |

#### 1.1 での召集軍関連法の変更

| 法律 | 変更前 | 変更後 |
|------|--------|--------|
| 民兵法（Common Militias） | 召集軍戦闘効率 +25% | **+10%** |
| 貴族召集軍（Noble Levies） | 召集量 +20% / 戦闘効率 +25% | **召集量 +50%** / 戦闘効率補正削除 |

#### 召集軍最大化のポイント

- 人口増加と食料確保を優先
- 各階級満足度 25% 以上を維持
- 序盤は数合わせ、後半は正規中心へ移行
- 1.1.10 以降は召集軍まわりの不具合が減り、再現性は上がった

### 統合軍備（Combined Arms）ボーナス

Patch 1.1 で追加された多兵科混成ボーナス。異なる兵科を一定割合以上編成に組み込むことで追加戦闘修正が発生する。ブランデンブルク/プロイセンの場合、歩兵（後半は擲弾兵）・騎兵・砲兵の三兵科混成が統合軍備ボーナスを最大限に活用できる編成となる。

序盤は召集軍主体で兵科バランスを取りにくいが、正規軍への移行が進む中盤以降は意識して混成比率を調整する。歩兵 5：騎兵 2：砲兵 1 前後を基準に、地形と敵編成に応じて微調整するのが安定した指針。

## 外交・同盟

### 序盤の優先関係

| 国/勢力 | 推奨 | 理由 |
|----------|------|------|
| デンマーク（DAN） | 同盟維持 | 開始時同盟。相手の戦争に参戦して**好感度（Favor）**を溜める |
| マイセン（MEI） | 同盟締結 | 南方パートナー。序盤の安全保障を厚くする |
| 皇帝（オーストリア等） | 関係維持 | ランク昇格・プロイセン王国化交渉に直結 |
| ポンメルン等の北方小国 | 征服対象 | 弱い相手から刻んで領土を広げる |
| ドイツ騎士団（TEU） | 征服対象 | ダンツィヒ確保でプロイセン地域の足場を作る |

> デンマークやマイセンは自前で戦争処理できることが多い。こちらは便乗参戦で好感度を稼ぎ、召集軍を温存すると安定する。

### 神聖ローマ帝国内の外交

皇帝を目指すかどうかは「皇帝を目指すべきか」（中盤戦略 > 神聖ローマ帝国の活用）を参照。基本方針は以下の通り。

- 帝国内の過度な拡張は**敵対心**急増から包囲網につながる
- CB なし宣戦は 1.1 以降さらにペナルティが重い
- 友好的な他国を皇帝にしたほうが交渉しやすい
- 選帝侯としての票は外交カードとして活用する

## 内政・経済

### インフラ建設の優先順位

| 項目 | やること |
|------|---------|
| 支配度 | **内閣（Cabinet）**を首都から順に動かし、中核州の支配度を回復 |
| 経済 | 市場建設と交易路開設で収入を安定化 |
| 属国管理 | 東部州は属国に委任。直轄化はディザスター後 |

### 階級（Estate）管理

| 項目 | やること |
|------|---------|
| 全般 | 各**階級（Estate）**の満足度 25% 以上を維持 |
| 聖職者 | 権力を **20% 未満**に抑える（プロイセン建国条件） |
| 貴族 | 権力を **20% 超過**に保つ（プロイセン建国条件） |
| 庶民 | 「共同の土地（Communal Lands）」を剥奪し、君主力回復を優先 |

> **プロイセン建国条件として聖職者権力 < 20%、貴族権力 > 20% が必要**なので、序盤から意識して調整すること。
